## If a judge pushes back

- *"Is this hardcoded?"* → "Type any goal or question yourself — there's no script for it. Only a live model grounded in her Twin can answer."
- *"What if the API is down?"* → "It falls back to a deterministic Twin model — the demo never breaks. Watch the badge: `Live · OpenAI` vs `Local Twin model`."
- *"What does this cost to run?"* → "Pennies per session — about 4 cents for a full demo. Negligible at enterprise scale."

## Recovery moves (if something stalls live)

- Career GPS slow? Keep narrating the reasoning stream — it streams progressively.
- Chat hangs? Use the scripted golden question ("How do I become an AI Engineer?") — it's instant and never calls the API.
- Whole site down? Persona switching and the golden path still render from local data.
