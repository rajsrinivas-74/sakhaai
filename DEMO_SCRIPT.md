## If a judge pushes back

- Is this hardcoded? Type any goal or question yourself. There is no script for it. Only a live model grounded in her Twin and performance data can answer.
- Does it really use the appraisal? Yes. The live prompt includes every KPP and the manager feedback, and is instructed to ground each gap in that evidence and not invent generic ones.
- What if the API is down? It falls back to a deterministic Twin model. The demo never breaks. Watch the badge: Live OpenAI versus Local Twin model.
- What does it cost to run? Pennies per session, about four cents for a full demo. Negligible at enterprise scale.

## Recovery moves (if something stalls live)

- Career GPS slow? Keep narrating the reasoning stream. It streams progressively.
- Chat hangs? Use the scripted golden question, How do I move into AI Delivery Management. It is instant and never calls the API.
- Whole site down? Persona switching and the golden path still render from local data.
